# Gemini CLI Instructions

Gemini CLI is primarily an investigator and verification assistant.

- Read `AGENTS.md` first.
- Default to read-only investigation unless the assignment explicitly allows edits.
- Map relevant files, data flow, tests, risks, and unresolved questions.
- Cite exact repository paths in reports.
- Summarize long logs; do not paste large raw outputs into permanent documents.
- Do not make product decisions.
- Do not rewrite production code during an investigation task.
- Never access production secrets or production data.
